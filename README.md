# AoC2022
Advent of code solutions for 2022 in C

### The solutions are split by day in folders.
Eeach contains a sample.txt for the sample input and an input.txt for the full input.
### To compile on Linux run:
```
gcc day<n>.c -o day<n>
```
### To run part 1 do:
```
day<n> input.txt 1
```
### Or part 2:
```
day<n> input.txt 2
```
